Agent Prompt Builder

A structured prompt is the difference between an agent that wanders and one that does the job. Pick a starting preset, fill in the parts you need, and copy a clean, well-organized prompt. Nothing is sent anywhere; everything runs in your browser.

Identity

Give the agent a name and a clear role. The role is the single most load-bearing line in the prompt.

Objective

What is this agent for? List one or more concrete tasks. Be specific about the end state.

Capabilities & tools (optional)

List the tools the agent can call. Use snake_case names that match your real tool definitions. Marking a tool required tells the model it should not finish without using it.

Workflow

The step-by-step process the agent should follow. Ordered steps beat a wall of prose.

Rules

Hard guardrails. Keep them short and absolute. Always and Never rules are easier for a model to follow than soft suggestions.

Output format

Tell the agent exactly how to shape its final answer. A precise contract here is what makes the output parseable and predictable.

Bullet list JSON object Markdown report Concise prose Table

Examples (optional, recommended)

One or two worked examples teach the model more than a paragraph of instructions. Show an input and the ideal output.

Error handling

What the agent should do when a tool fails or returns nothing useful.
